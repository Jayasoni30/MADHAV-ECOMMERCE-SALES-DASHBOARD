# 📊 Madhav Store Sales Dashboard

## 📌 Overview
An interactive **Sales Dashboard** built using Power BI to analyze business performance across sales, profit, customers, and product categories.

## 🎯 Objective
* Track (sales & profit performance)
* Identify (top customers & products)
* Analyze (regional and category trends)
* Enable (data-driven decision making)

## 📊 Key Features
* 📈 KPI Cards (Sales, Profit, Quantity, AOV)
* 🌍 Sales by State Analysis
* 🛍️ Category & Sub-category Insights
* 📅 Monthly Profit Trends
* 👥 Top Customers Analysis
* 💳 Payment Mode Distribution

## 🛠️ Tech Stack
* Power BI
* Power Query
* DAX

## 💡 Key Insights
* Top states contribute majority of revenue
* Some sub-categories generate losses
* Sales show clear monthly trends
* Few customers drive most revenue

## 📎 Project Files
* `.pbix` Dashboard file
* Dataset (CSV/Excel)
